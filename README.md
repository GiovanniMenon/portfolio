## Tech stack


- Next.js
- [Shadcn/ui](https://ui.shadcn.com/)
  - Used for :
    - Card (_Project_ , _Contact_). 
    - Form (_label_, _textArea_, _input_).
    - NavBar Menu.
    - Tooltip.
    - Separator.
    - Button. 
    - Theme Toggle.
    - Themes and Colors [(here's)](https://ui.shadcn.com/themes).
- [Aceternity UI](https://ui.aceternity.com/)
  - Used for :
    - Grid Background.
    - Laser Beam Tracer.
- Tailwind.
